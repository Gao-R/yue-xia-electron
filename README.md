# electron-vue

> An electron-vue project

#### Build Setup

月下鹿 桌面应用版

普通用户和管理员用户，用户注册，登陆，找回，点赞，评论，用户发表图片，视频，文章，收藏小说，漫画，影视

使用electron+vue进行开发

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build

```
